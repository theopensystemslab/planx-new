PlanX /admin endpoints

Admin endpoints are available to developers and editors for viewing, downloading, and troubleshooting data about Planx services and sessions.

These endpoints are only accessible to users who are signed into the Editor, have a white-listed email address, and are connecting via a WARP client.

Endpoint Optional params Description
/feedback Downloads a FeedbackFish CSV
/session/:sessionId/summary Returns a passport, breadcrumbs, and other key details for a given session
/session/:sessionId/xml Generates a OneApp XML, relies on uniform_applications
/session/:sessionId/bops Generates a BOPS payload, relies on bops_applications
/session/:sessionId/csv ?download=true Generates or downloads an application CSV file
/session/:sessionId/csv-redacted ?download=true Generates or downloads an application CSV file with personal details redacted
/session/:sessionId/html Generates an application overview HTML
/session/:sessionId/html-redacted Generates an application overview HTML with personal details redacted
/session/:sessionId/zip ?includeXML=true Generates a zip file for Send to Email, or Uniform when XML is included
